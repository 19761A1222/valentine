### Adding Photos to Memories Page

1. Create a folder called `images` in your GitHub repository
2. Upload your photos (click "Add file" â†’ "Upload files")
3. Name them: photo1.jpg, photo2.jpg, etc.
4. Edit `memories.html`:
   - Find the line: `<!-- <img src="images/photo1.jpg" alt="Our first date"> -->`
   - Remove the `<!--` and `-->` to uncomment it
   - Update the descriptions, titles, and dates!

### Customizing the Quiz

Edit `quiz.html`:
1. Find the `correctAnswers` object in the code
2. Change the letters to match your correct answers
3. Update the question text and options to your own questions!

### Personalizing the Timeline

Edit `timeline.html`:
1. Find each timeline item
2. Update the dates, titles, and descriptions
3. Add more timeline items by copying the structure!

### Editing Reasons

Edit `reasons.html`:
1. Change the reason text in each card
2. Add more reason cards by copying the structure!
